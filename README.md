# 📁 Мои Документы

Современное и безопасное Android-приложение для локального хранения личных документов и важных данных. Создано с акцентом на приватность, удобство использования и современный дизайн (Material Design 3).

## ✨ Главные возможности

* **🔒 Высокий уровень приватности:** Вход в приложение защищен PIN-кодом и биометрией (отпечаток пальца/FaceID). Все данные хранятся локально на устройстве (без облачных серверов).
* **🗂 Умная организация:** Создание пользовательских папок для документов. Поддержка интуитивного Drag & Drop для изменения порядка папок и свайпов для их редактирования.
* **🔍 Продвинутый поиск:** Мгновенный поиск документов по названию, типу или содержимому полей прямо из удобной панели навигации.
* **🎨 Современный интерфейс:** Плавные анимации переходов, динамическая нижняя панель навигации в стиле Material Expressive и поддержка темной темы.

## 🛠 Технологический стек

Приложение разработано с использованием современных инструментов Android-разработки:
* **Язык:** [Kotlin](https://kotlinlang.org/)
* **UI-фреймворк:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
* **Архитектура:** MVVM (Model-View-ViewModel)
* **Локальная база данных:** [Room Database](https://developer.android.com/training/data-storage/room)
* **Навигация:** Compose Navigation
* **Асинхронность:** Coroutines & StateFlow

## 🚀 Установка

Вы можете установить готовое приложение на свой Android-смартфон без необходимости собирать его из исходного кода:
1. Перейдите в раздел [Releases](../../releases) в правой части страницы.
2. Скачайте последнюю версию файла `DocumentsApp-vX.X.apk`.
3. Откройте файл на вашем смартфоне и разрешите установку из неизвестных источников.
